== bitewolf

Project status: Active, unfinished

What is bitewolf?

1.) Create an account. Create a lunch Group for your coworkers, friends, etc.
2.) Create a Lunch. Specifies a date and time.
3.) Group members suggest lunch Location, "chipotle - eat in," etc.
4.) Group members vote on suggested Locations.
5.) Lunch Location is chosen by simple majority vote, or is chosen at random upon a tie.
